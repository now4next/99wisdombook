## 🎯 현재 구현 상태

### 플로팅 TOC 버튼 통합
모바일과 PC 모두에서 **동일한 플로팅 원형 버튼**을 사용하고 있습니다.

#### 모바일 (768px 이하)
```css
#floating-toc-button {
  display: flex !important;  /* 표시 */
  position: fixed;
  top: auto;
  bottom: 20px;              /* 하단 위치 */
  right: 20px;
  width: 50px;               /* 50×50px */
  height: 50px;
  background: linear-gradient(135deg, #666 0%, #888 100%);
  border: 2px solid #999;
  border-radius: 50%;        /* 원형 */
  z-index: 9998;
}

#floating-toc-button svg {
  width: 24px;
  height: 24px;
}
```

**위치**: 우측 하단 (bottom: 20px, right: 20px)
**크기**: 50×50px
**모양**: 원형 (햄버거 아이콘 ≡)

#### PC (769px 이상)
```css
#floating-toc-button {
  display: flex !important;  /* 표시 */
  position: fixed;
  top: 65px;                 /* 상단 위치 */
  right: 30px;
  width: 56px;               /* 56×56px */
  height: 56px;
  background: linear-gradient(135deg, #666 0%, #888 100%);
  border: 2px solid #999;
  border-radius: 50%;        /* 원형 */
  z-index: 9998;
}

#floating-toc-button svg {
  width: 28px;
  height: 28px;
}
```

**위치**: 우측 상단 (top: 65px, right: 30px)
**크기**: 56×56px
**모양**: 원형 (햄버거 아이콘 ≡)

## 💡 버튼 동작

### 모든 화면 크기에서 동일
1. **버튼 클릭/터치**: TOC 패널 열림
2. **TOC 패널**: 우측에서 슬라이드 인
3. **챕터 선택**: 해당 위치로 스크롤
4. **닫기**: X 버튼 또는 오버레이 클릭
5. **버튼 재클릭**: TOC 패널 닫힘

### 호버 효과 (PC)
```css
#floating-toc-button:hover {
  transform: scale(1.1);
  background: linear-gradient(135deg, #777 0%, #999 100%);
  box-shadow: 0 6px 20px rgba(0, 0, 0, 0.3);
  border-color: #aaa;
}
```
