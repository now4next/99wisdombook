---

## ✅ 해결 방법

### 최종 솔루션: 브라우저 내장 번역 안내

Google Translate URL 방식 대신, **브라우저 내장 번역 기능**을 사용하도록 안내하는 방식으로 변경했습니다.

---

## 🎯 작동 방식

### 1. 언어 메뉴 클릭
사용자가 상단 언어 메뉴에서 원하는 언어를 클릭합니다.

### 2. 번역 안내 오버레이 표시
아름다운 오버레이가 나타나며 **3가지 번역 방법**을 안내합니다:

#### 📱 방법 1: Chrome/Edge 브라우저
- 주소창 오른쪽의 **번역 아이콘 (🌐)** 클릭
- 또는 페이지에서 **우클릭** → **"한국어 번역"** 선택
- 언어를 원하는 언어로 선택

#### 🧭 방법 2: Safari
- 주소창의 **aA** 아이콘 클릭
- **"Translate to [언어]"** 선택

#### 🌐 방법 3: Google Translate 앱
- Google Translate 앱 실행
- **"카메라"** 또는 **"이미지"** 기능 사용
- 화면을 스캔하여 번역

### 3. 브라우저가 자동으로 번역
사용자가 브라우저 기능을 사용하면:
- ✅ 전체 페이지(99개 챕터) 즉시 번역
- ✅ 원본 디자인 완벽 유지
- ✅ 지역 제한 없음

## ❓ FAQ

### Q1. Chrome에서 번역 아이콘이 안 보여요
**A**: 페이지를 새로고침하면 주소창 오른쪽에 번역 아이콘(🌐)이 나타납니다.

### Q2. Safari에서 aA 아이콘이 없어요
**A**: Safari 14 이상 버전에서만 지원됩니다. 최신 버전으로 업데이트하세요.

### Q3. 모바일에서도 작동하나요?
**A**: 네! Chrome, Safari 모바일 브라우저 모두 번역 기능을 지원합니다.

### Q4. 번역 품질은 어떤가요?
**A**: 브라우저의 번역 엔진은 Google Neural MT를 사용하므로 매우 높은 품질입니다.

### Q5. 비용이 드나요?
**A**: 완전 무료입니다! 브라우저의 내장 기능을 사용합니다.
