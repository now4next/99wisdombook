🌐 자동 번역 기능 안내

방대한 책 내용(99개 챕터)을 효율적으로 번역하는 3가지 방법을 소개합니다.

✨ 방법 1: 언어 메뉴 클릭 (추천!)

가장 간단하고 효과적인 방법

  1. 📖 책 페이지 상단의 언어 메뉴를 확인하세요
  2. 🌐 원하는 언어를 클릭하세요 (예: English, 日本語, 中文)
  3. ✅ Google Translate가 자동으로 전체 페이지를 번역합니다
  4. 🔄 한국어로 돌아가려면 한국어 버튼 클릭

장점:

  • ✅ 클릭 한 번으로 전체 페이지 번역
  • ✅ 99개 챕터 모두 자동 번역
  • ✅ 이미지 캡션도 번역됨
  • ✅ 원본 디자인 유지

🌐 방법 2: 브라우저 자동 번역

Chrome, Edge, Safari의 내장 번역 기능 사용

Google Chrome / Microsoft Edge:

  1. 📄 책 페이지를 엽니다
  2. 🖱️ 주소창 오른쪽의 번역 아이콘 클릭
  3. 🌐 또는 우클릭 → "한국어 번역" 선택
  4. ⚙️ 또는 Ctrl+Shift+P (Mac: Cmd+Shift+P)

Safari (Mac/iPhone):

  1. 📱 주소창의 aA 아이콘 클릭
  2. 🌐 "Translate to [언어]" 선택

지원 브라우저:

🌐 Chrome 🔷 Edge 🧭 Safari

🔗 방법 3: Google Translate 직접 사용

Google Translate 웹사이트에서 URL 번역

  1. 🌐 translate.google.com 접속
  2. 📋 "웹사이트" 탭 클릭
  3. 🔗 책 페이지 URL 입력:
    https://now4next.github.io/99wisdombook/book.html
  4. 🌐 원하는 언어 선택 (예: 영어, 일본어, 중국어)
  5. ✅ 번역된 페이지 보기

❓ 자주 묻는 질문 (FAQ)

Q1. 번역 품질은 어떤가요?

Google Translate는 최신 AI 기술(Neural Machine Translation)을 사용하여 매우 높은 품질의 번역을 제공합니다. 99개 챕터의 방대한 내용도 문맥을 이해하여 자연스럽게 번역합니다.

Q2. 번역 속도는 빠른가요?

네! Google Translate는 전체 페이지를 1-2초 내에 번역합니다. 99개 챕터를 모두 한 번에 번역하므로 매우 효율적입니다.

Q3. 디자인이 깨지지 않나요?

Google Translate는 원본 HTML 구조를 유지하면서 텍스트만 번역하므로, 아름다운 원본 디자인이 그대로 유지됩니다.

Q4. 비용이 드나요?

완전 무료입니다! Google Translate는 무료 서비스이며, 제한 없이 사용할 수 있습니다.

Q5. 모바일에서도 작동하나요?

네! 모든 방법이 모바일에서도 완벽하게 작동합니다. 특히 방법 1(언어 메뉴 클릭)은 모바일에서 가장 편리합니다.
