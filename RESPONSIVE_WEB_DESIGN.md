## 🎯 구현 세부사항

### **CSS 미디어 쿼리**

```css
/* 태블릿 (768px ~ 1024px) */
@media screen and (max-width: 1024px) {
  body {
    padding: 30px 40px;
    font-size: 15px;
  }
}

/* 모바일 (768px 이하) */
@media screen and (max-width: 768px) {
  body {
    padding: 20px 16px;
    padding-top: 100px;
    font-size: 14px;
  }
  
  #language-selector {
    flex-direction: column;
    gap: 8px;
  }
  
  #user-info {
    width: 100%;
    justify-content: space-between;
  }
  
  #language-links {
    width: 100%;
    flex-wrap: wrap;
    justify-content: center;
  }
}

/* 소형 모바일 (480px 이하) */
@media screen and (max-width: 480px) {
  body {
    padding: 16px 12px;
    font-size: 13px;
  }
}

/* 가로 모드 */
@media screen and (max-width: 768px) and (orientation: landscape) {
  #language-selector {
    flex-direction: row;
  }
}

/* 대형 화면 (1440px+) */
@media screen and (min-width: 1440px) {
  body {
    max-width: 1000px;
    font-size: 17px;
  }
}
```

### **Viewport 설정**

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=5.0, user-scalable=yes">
```

**설명**:
- `width=device-width`: 디바이스 너비에 맞춤
- `initial-scale=1.0`: 초기 줌 레벨 100%
- `maximum-scale=5.0`: 최대 500% 확대 가능
- `user-scalable=yes`: 사용자 줌 허용 (접근성)
